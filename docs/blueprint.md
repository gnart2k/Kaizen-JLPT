# **App Name**: Nihongo Compass

## Core Features:

- Interactive Dashboard: Provides a summarized view of learning stats, daily progress, and quick access to practice modes, mock tests, and analytics, with a JLPT level selector.
- Practice Mode: Allows users to practice vocabulary, grammar, and listening skills with multiple-choice questions and immediate feedback.
- AI-Powered Feedback: Provides detailed explanations on why answers are correct or incorrect, suggests related study links, and highlights relevant grammar rules or vocabulary using the Gemini API tool.
- Mock Test Simulation: Offers full-length, timed mock JLPT tests with vocabulary, grammar, and listening sections, providing a detailed result summary.
- Performance Analytics: Tracks historical test data, visualizes strengths and weaknesses with radar charts, and monitors study streaks and improvement trends, stored and retrieved from PostgreSQL via Drizzle.
- Vocabulary and Grammar Library: Enables users to browse searchable word and grammar lists by JLPT level, mark items as mastered or to review, and utilize flashcard mode with spaced repetition.
- Personalized Learning: Uses Gemini to explain meanings and usage context, enhancing vocabulary and grammar understanding with AI assistance.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to provide a calm, focused atmosphere conducive to learning.
- Background color: Light gray (#F5F5F5), offering a clean and distraction-free backdrop that supports prolonged screen time and visual comfort.
- Accent color: Soft lavender (#C5CAE9), creating a subtle visual interest and highlighting key interactive elements without overwhelming the monochrome scheme.
- Body and headline font: 'Inter', a clean and rounded sans-serif for readability.
- Utilize simple, line-based icons that complement the rounded corners and soft shadows, ensuring a consistent visual language.
- Employ rounded corners (rounded-2xl), soft shadows (shadow-md), and ample spacing (p-4, p-6) to create a modern and comfortable user interface.
- Integrate subtle animations using Framer Motion to enhance user engagement without causing distraction.
